## 7. ESPECIFICAÇÕES TÉCNICAS

### Formato de Entrega
- **Formato:** WebP (primário), PNG (fallback)
- **Qualidade:** 85-90%
- **Perfil de Cor:** sRGB
- **Resolução:** Gerar em 2x para retina

### Nomenclatura dos Arquivos
```
tupan-{id}.webp
tupan-{id}@2x.webp
```

| Nome | Hex | Uso |
